2) open regedit
3) Go to "Computer\HKEY_CLASSES_ROOT\Directory\Background\shell\"

4) Create new key called "cexplore"
5) Under that, create a new key called "command"

6) In "cexplore" edit default string to -> Open in CreatorExplorer
7) add new string called "Icon" and paste path of .exe file.

8) In "command" edit default string to -> "PathOfEXE.exe" "%v" "PathToEXEWith/"
